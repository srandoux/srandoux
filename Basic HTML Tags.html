Ce qui suit n'est pas obligaoire en HTML5. Mais, il est fortement recommandé de:
  • écrire les tags en minuscule
  • tous les tags doivent être suivis de leur tags de fin respectifs par exemple un paragraphe doit être délimité par <p> ... </p> sauf les tags vides tels <hr> et <br>
  • tous les attributs de tags doivent mis entre guillemets <input type="text" name="nom">
  • Les delimiter de tags doivent être dans l'ordre, par exemple
    <p><strong><em>Mon paragraph en gras</em></strong></p>

Qui plus est, il faut indenter de façon appropriée pour rendre le code lisible est facile à maintenir.

Introduction

HTML (HyperText Markup Language) est un "markup" langage qui insert des "tags" ou balises en Français dans un document texte. Le navigateur interprète ces "markups tags" pour formater le document et l'afficher. Le nagivateur décide la mise en page car les nagivateurs servent sur de nombreux systèmes, ordinateur, télévision, téléphone...

Le navigateur affiche le document en fonction de critères implicites et d'autres explicites. Certains de ces critères sont:

  • Les markup tags utilisés.
  • La taille physique de page.
  • Les polices de caractères utilisées

Le navigateur ignoree les espaces supplémentaires et les nouvelles lignes entre les mots et les markups tags quand il lit le document. Ainsi les textes suivant seront formatés de façon identique.

Le contenu html commence par le tag <html> et fini par </html>. Le document est habituellement découpé en 2 sections:

  • head, encadré par <head> et </head>
  • body, encadré par <body> et <body>

La section head d'un document HTML contient des informations d'identication et de controle qui ne sont pas affichées par le navigateur. Comme par exemple un titre entouré par les <title> et <title> tags.

La section body du document et celle qui sera affichée par le navigateur. Dans cette section, différents tag serviront à la mise en page du texte.

Le langage markup est composé de tags comme <b> qui indique que le texte qui suit est en gras. Ce formatage est désactivé par le tag inverse </b>.

La mise en page de base d'un document HTML et l'affichage dans un navigateur comme Firefox, Chrome, Safari, Edge est comme ceci:
